Ubuntu Tasks
============

### Installation ###

**Dependencies:**

 * Ubuntu SDK
 * U1db - qtdeclarative5-u1db1.0
 * QtSystemInfo - qtdeclarative5-systeminfo-plugin

To run, simply open in Qt Creator/Ubuntu SDK, and click the run button!
